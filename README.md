# Xin chào, mình đang học Python

Mình đang xây dựng nền tảng Python từ các bài cơ bản đến những game 2D nhỏ. Mục tiêu hiện tại của mình là viết code để hiểu thật sự: biết biến, điều kiện, vòng lặp, hàm, file, class, object, và cách ghép chúng thành một chương trình có thể chạy được.

## Mình đang học

- Python cơ bản: biến, kiểu dữ liệu, `input()`, `print()`, `if`, `for`, `while`
- Cấu trúc dữ liệu: `list`, `tuple`, `dictionary`, mảng 2 chiều
- Hàm và cách tách code thành từng phần nhỏ
- Đọc và ghi file text
- Xử lý lỗi với `try`, `except`
- Class và object
- Làm game 2D cơ bản bằng Python

## Dự án nhỏ của mình

| Dự án | Mô tả |
| --- | --- |
| Calculator | Máy tính cộng, trừ, nhân, chia có xử lý toán tử |
| Translator | Từ điển đơn giản dịch từ tiếng Anh sang tiếng Việt |
| Phone Book | Đọc và ghi danh bạ bằng file text |
| Quiz Game | Game trắc nghiệm có câu hỏi, đáp án và tính điểm |
| Flappy Bird | Game chim bay qua ống, có tính điểm và game over |
| Evo Wars Arena | Game đấu kiếm 2D có bot, skin, level và bảng điểm |
| Slither Arena | Game rắn đấu trường có bot, thức ăn, tăng tốc và bảng điểm |

## Kỹ năng đang luyện

```python
def learning_python():
    basics = ["variables", "if else", "loops", "functions"]
    projects = ["calculator", "quiz", "translator", "2d games"]

    for skill in basics:
        print("Đang luyện:", skill)

    return projects
```

## Cách mình học

Mình thích học bằng cách làm các chương trình nhỏ trước, sau đó nâng cấp dần:

1. Viết bản chạy được trước
2. Đọc lại từng dòng code
3. Tìm lỗi và sửa lỗi
4. Thêm tính năng mới
5. Tách code gọn hơn bằng hàm và class

## Mục tiêu tiếp theo

- Viết code rõ ràng hơn, đặt tên biến dễ hiểu hơn
- Làm game có menu, cài đặt, bot và bảng điểm
- Hiểu sâu hơn về class, object và game loop
- Học cách lưu dữ liệu bằng file JSON
- Đưa các bài tập thành những project hoàn chỉnh hơn

## Ghi chú
